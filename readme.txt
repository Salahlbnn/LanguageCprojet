Jeu Mathématique en C

Description

Ce projet est un petit jeu éducatif développé en langage **C**, destiné à entraîner les utilisateurs aux **opérations de base** (addition, soustraction, multiplication, tables de multiplication).  
Le jeu permet de suivre la progression de l'utilisateur en enregistrant son score dans un fichier `scores.txt`.

---

 Fonctionnalités

- Menu interactif avec plusieurs choix :
  - Addition
  - Soustraction
  - Multiplication
  - Tables de multiplication
  - Division (à venir)
Trois tentatives autorisées pour chaque opération
Attribution de points selon la performance
Lecture et mise à jour du score dans un fichier texte
 Enregistrement de l’heure et la date du score
